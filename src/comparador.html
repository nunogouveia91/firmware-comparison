ℹ️ O que é esta tab?

Compara as médias de cada indicador entre FW-A (Control/Antes) e FW-B (Upgrade/Depois) num único snapshot.

Cada linha mostra o valor médio de cada grupo, a variação percentual (Δ%) e a classificação automática:

  • 🔴 Blocker / Major — degradação acima dos limiares configurados
  • 🟡 Atenção — degradação moderada
  • 🟢 Good — melhoria além do limiar positivo
  • ✅ OK — dentro do intervalo aceitável
  • S/D — sem dados num ou ambos os grupos

Útil para uma visão geral rápida do impacto do firmware em todos os indicadores.

ℹ️ O que é esta tab?

Mostra a evolução temporal de cada indicador para FW-A e FW-B ao longo do tempo, num gráfico de linha.

Dois modos de visualização:

  • Períodos Cruzados (default) — alinha os dois períodos por posição relativa (+0h, +4h…). Ideal para cenários antes/depois onde os timestamps não se sobrepõem.
  • Períodos Separados — cada grupo é plotado nos seus timestamps reais num eixo X partilhado.

O Δ% exibido usa sempre o alinhamento posicional. Útil para detectar tendências e padrões de degradação ao longo do tempo.
